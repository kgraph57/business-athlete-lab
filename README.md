# Business Athlete Lab

**働く身体をアップデート。ランチタイムで読む健康戦略。**

https://kgraph57.github.io/business-athlete-lab

---

エビデンスに基づく、ビジネスアスリートのための健康マガジン。医師として臨床に携わりながら CrossFit、HYROX、トライアスロンに挑戦する筆者が、国内外の最新論文を読み解き、エビデンスベース戦略を届けるメディアです。

## Topics

| テーマ | Topic |
|--------|-------|
| 運動 | Exercise |
| 睡眠 | Sleep |
| 栄養 | Nutrition |
| メンタル | Mental |
| 生活習慣 | Lifestyle |
| 長寿 | Aging / Longevity |
| 総合 | Integrative |

## Tech Stack

- **Framework**: Next.js 16 (Static Export)
- **Styling**: Tailwind CSS 4
- **Search**: Fuse.js
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions
